css3新属性

Nov 8
一、css3边框：border-radius/box-shadow/border-image
code
1 border-radius:10px/20%
2 border-top-right-radius:10px/20%
3 border-top-left-radius:10px/20%
4 border-bottom-left-radius:10px/20%
5 border-bottom-right-radius:10px/20%
6 border-radius:10px 10px 10px 10px(四个值是左上，右上，右下，左下)
7 border-radius:10px 10px(两个个值是左上，右下；右上，左下)
二、css3背景：background-image/background-size/background-origin/background-clip
code
1 background-image:url(a.jpg,b.jpg)
2 background-position: right bottom, left top;/cover/contain
3 background-repeat: no-repeat/repeat-x/repeat-y
4 background-size: 10px 10px/20% 20%/contain/cover
5 background-origin: border-box/padding-box/content-box(背景图片的位置)
6 background-clip: border-box/padding-box/content-box(背景剪裁属性是从指定位置开始绘制。)
三、css3渐变：background: linear-gradient(red, blue);支持ie10以上的浏览器

1、线性渐变（Linear Gradients）- 向下/向上/向左/向右/对角方向

2、径向渐变（Radial Gradients）- 由它们的中心定义

code
1 background: linear-gradient(red, blue); /* 默认情况下是从上往下 */
2 background: linear-gradient(to left, red , blue); /* left,right,top,bottom
3 background: linear-gradient(to bottom right, red , blue); /*右下*/
4 background: linear-gradient(180deg, red, blue); /* 按一定的角度渐变 */
5 background: linear-gradient(to right, rgba(255,0,0,0), rgba(255,0,0,1)); /*透明度的渐变 */
6 background: radial-gradient(circle red 5%, green 10%, blue 20%); /* 标准的语法 */
7 background: repeating-radial-gradient(red, yellow 10%, green 15%);/*重复的径向渐变*/
四、css3文本效果

1、text-shadow:第一个参数左右距离，第二个参数上下距离，第三个参数模糊程度，第四个参数颜色

2、box-shadow:第一个参数左右距离，第二个参数上下距离，第三个参数延伸长度，第四个参数阴影模糊程度，第五个参数颜色

也可以通过为元素来添加阴影

3、text-overflow：ellipsis(显示省略号)，clip(超出裁剪)这个属性通常与title属性搭配使用

五、css3字体

1、@font-face

六、css3之2D旋转：translate(10px 20px),rotateXYZ(30deg),scale(2,3),skew(20deg,30deg)
七、css3之过渡

transition:第一个参数过渡属性，第二个参数过渡花费时间，第三个属性过渡方式，第四个参数过渡延迟

八、css3之动画

1、@keyframes规则是创建动画。 @keyframes规则内指定一个CSS样式和动画将逐步从目前的样式更改为新的样式。

2、animation动画持续时间

九、css3之多列属性

column-count:列数；column-gap：列与列之间的间隙；column-rule-style:列边框样式；column-rule-width列边框厚度；column-rule-color: 列边框颜色；column-width：列宽

十、css3之resize在ie中不兼容：任何标签都有这个属性，不只是textarea标签
十一、css3之border-sizing属性值：border-box/content-box
十二、css之outline类似于border，但是在border外层，通过outline-offset可以设置border与outline的间隙
十三、css3之filter属性图片滤镜
十四、css之button的属性值：cursor：not-allowed禁用按钮
十五、flex布局支持ie11以上的浏览器，弹性布局只是针对弹性子元素

1、direction可以指定布局方向

2、flex-wrap：nowrap/wrap

十六、媒体查询@media支持ie9以上

1、屏幕类型：screen，print，speech，all

code
1 @media screen and (max-width: 699px) and (min-width: 520px) {
2 ul li a {
3 padding-left: 30px;
4 background: url(email-icon.png) left center no-repeat;
5 }
6 }
